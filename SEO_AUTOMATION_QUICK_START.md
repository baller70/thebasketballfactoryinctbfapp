## 📞 TROUBLESHOOTING

### Email Reports Not Arriving?
1. Check spam/promotions folder
2. Verify Resend API is active
3. Check `SEOAuditLog` for send status

### Rankings Not Improving?
1. Wait 4-6 weeks (SEO takes time)
2. Check weekly reports for trends
3. Review AI content suggestions
4. Verify GA sync is working

### Social Posts Not Publishing?
1. Verify Twitter/X credentials in `/home/ubuntu/.config/abacusai_auth_secrets.json`
2. Check `SEOAuditLog` for posting attempts
3. Ensure API limits not exceeded

### Daemon Tasks Not Running?
1. Use `list_daemon_tasks` command to check status
2. Verify tasks show as ACTIVE
3. Check next run times are in future
