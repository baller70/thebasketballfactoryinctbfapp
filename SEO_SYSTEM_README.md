# SEO Content Management System

## Overview

The Basketball Factory Inc now has a comprehensive **SEO Content Management System** powered by Abacus.AI's advanced language models. This system automatically generates, optimizes, and manages SEO content across all website pages to improve search engine rankings for local basketball training services in Sparta, NJ and surrounding areas.

## Key Features

### 1. **AI-Powered Content Generation**
- Leverages Abacus.AI's language models to create human-like, undetectable AI content
- Generates SEO-optimized titles, meta descriptions, headings, and page content
- Focuses on local SEO for Sparta, NJ and Sussex County areas
- Targets parents of children aged 7-18 interested in basketball training

### 2. **Keyword Management**
- Track and manage target keywords with priorities
- Monitor search volume and difficulty scores
- Categorize keywords by type (local, training, programs, etc.)
- Link keywords to specific pages for targeted optimization

### 3. **Page Configuration**
- Configure SEO settings for each page individually
- Set update frequencies (daily, weekly, biweekly, monthly)
- Enable/disable auto-generation per page
- Define local area targets for each page

### 4. **Content Versioning & Approval**
- Version control for all generated content
- Manual review and approval workflow
- Publish/unpublish content versions
- A/B testing capabilities for content variations

### 5. **Performance Tracking**
- Monitor impressions, clicks, CTR, and search positions
- Track conversions and organic traffic
- Analyze performance by keyword, page, and date range
- Identify top performers and underperforming content

### 6. **Analytics & Reporting**
- AI-generated weekly/monthly analytics reports
- Performance insights and recommendations
- Competitive analysis suggestions
- Actionable items prioritized by impact

### 7. **Audit Logging**
- Complete history of all SEO changes
- Track who made changes and when
- Monitor content generation and publication events
- Troubleshoot issues with detailed logs

## Access the SEO Dashboard

1. Log in to the admin panel at `/admin` using your credentials
2. Click on "SEO Management" in the sidebar navigation
3. You'll see the main SEO dashboard with overview metrics

## How to Use

### Managing Keywords

**To Add a New Keyword:**
1. Navigate to **Admin → SEO Management → Keywords**
2. Click "Add Keyword" button
3. Enter keyword details:
   - Keyword phrase (e.g., "basketball training sparta nj")
   - Category (e.g., "local-primary", "youth-programs")
   - Target location (e.g., "Sparta, NJ")
   - Search volume (monthly estimates)
   - Difficulty score (1-100)
   - Priority (1-10, where 10 is highest)
4. Click "Add Keyword"

**Managing Existing Keywords:**
- Click "Edit" to modify keyword settings
- Toggle "Active/Inactive" to enable/disable tracking
- Click "Delete" to remove a keyword (use with caution)

### Configuring Pages for SEO

**To Configure a Page:**
1. Navigate to **Admin → SEO Management → Pages**
2. Click "Add Page Configuration" or select an existing page
3. Configure settings:
   - Page path (e.g., "/", "/private-lessons")
   - Page friendly name
   - Target keywords (select from your keyword list)
   - Local area targets (cities to focus on)
   - Update frequency
   - Content strategy guidelines
   - Enable/disable auto-generation
4. Save the configuration

### Generating SEO Content

**Automatic Generation:**
1. Ensure page configuration has "Auto Update Enabled"
2. System will generate content according to update frequency
3. Generated content enters "Pending Review" status

**Manual Generation:**
1. Go to **Admin → SEO Management → Generate**
2. Select the page you want to generate content for
3. Click "Generate Content"
4. Wait for AI to generate optimized content (typically 10-30 seconds)
5. Review the generated content

### Reviewing & Publishing Content

1. Navigate to **Admin → SEO Management → Content Review**
2. View all pending content versions
3. For each version:
   - Click to preview the generated content
   - Review title, meta description, headings, and content blocks
   - Check SEO score and readability score
   - Add review notes if needed
4. Actions available:
   - **Approve**: Mark content as approved (doesn't publish yet)
   - **Reject**: Reject the content with a reason
   - **Publish**: Immediately publish content to the live site
   - **Request Changes**: Add notes for regeneration

**Publishing Workflow:**
- By default, generated content requires manual approval
- Once approved, you can publish it to make it live
- Publishing automatically unpublishes other versions of the same page
- You can unpublish content anytime

### Viewing Analytics

**Dashboard Overview:**
1. Main SEO dashboard shows 30-day performance summary
2. View metrics:
   - Total impressions
   - Total clicks
   - Average CTR
   - Average search position
   - Conversions

**Top Keywords:**
- See best performing keywords with click data
- Analyze CTR and conversion rates
- Identify opportunities for improvement

**Performance Reports:**
1. Navigate to **Admin → SEO Management → Reports**
2. Generate new reports:
   - Weekly reports (7 days)
   - Monthly reports (30 days)
   - Custom date ranges
3. View AI-generated insights and recommendations
4. Export reports for stakeholder sharing

### Recording Performance Data

Performance data can be entered manually or integrated with Google Search Console:

**Manual Entry:**
1. Go to **Admin → SEO Management → Performance**
2. Click "Add Performance Data"
3. Enter metrics for specific keywords/pages:
   - Date
   - Keyword
   - Page path
   - Impressions
   - Clicks
   - Average position
   - Organic traffic
   - Conversions

**Recommended Integration:**
- Connect Google Search Console for automatic data import
- Configure in Settings → Integration Settings

### Configuring Global Settings

1. Navigate to **Admin → SEO Management → Settings**
2. Configure:
   - **Auto-Generation Settings**
     - Enable/disable automatic content generation
     - Default update frequency
     - Quality threshold for auto-publish
     - Require manual approval
   - **Geographic Targeting**
     - Primary location (default: Sparta, NJ)
     - Target radius in miles
     - Target cities list
     - Target counties
   - **Content Generation**
     - Content tone
     - Brand voice guidelines
     - Target audience description
     - Key selling points
   - **Keywords**
     - Primary keywords focus
     - Secondary keywords
     - Keywords to exclude
   - **Performance Thresholds**
     - Minimum impressions
     - Target CTR
     - Target search position
     - Alert thresholds
   - **Reporting**
     - Enable weekly/monthly reports
     - Report recipients (email addresses)
     - Alert recipients

## Best Practices

### Content Generation:
1. **Review Before Publishing**: Always review AI-generated content
2. **Maintain Brand Voice**: Ensure content aligns with your brand
3. **Local Relevance**: Verify local references are accurate
4. **Keyword Density**: Aim for natural integration, not keyword stuffing
5. **Readability**: Target score above 60 for parent audience

### Keyword Strategy:
1. **Start Local**: Focus on "Sparta, NJ" + service keywords
2. **Expand Gradually**: Add nearby cities as you gain traction
3. **Monitor Performance**: Track which keywords drive conversions
4. **Competitive Analysis**: Research competitor keywords
5. **Long-Tail Keywords**: Include specific, less competitive phrases

### Performance Monitoring:
1. **Weekly Reviews**: Check dashboard weekly for trends
2. **Monthly Reports**: Generate monthly reports for deeper insights
3. **Act on Insights**: Implement AI recommendations
4. **A/B Testing**: Test different content variations
5. **Adjust Strategy**: Pivot based on performance data

### Update Frequency:
- **Homepage**: Weekly (high-traffic, competitive)
- **Program Pages**: Biweekly to monthly
- **Seasonal Pages**: Before each season starts
- **Static Pages**: Monthly or as needed

## Troubleshooting

### Content Generation Fails:
1. Check API key is configured (ABACUSAI_API_KEY in .env)
2. Verify page configuration has target keywords
3. Check audit logs for error messages
4. Ensure local area targets are set

### Content Not Publishing:
1. Verify content status is "approved"
2. Check if manual approval is required in settings
3. Ensure SEO score meets threshold (default: 70)
4. Review audit logs for errors

### Performance Data Not Showing:
1. Ensure data is being recorded (check Performance page)
2. Verify date range in queries
3. Check keyword associations are correct
4. Review database connection

### Auto-Generation Not Working:
1. Check "Auto Update Enabled" in page config
2. Verify "Auto Generate Enabled" in global settings
3. Check nextScheduledUpdate date
4. Review audit logs for generation attempts
